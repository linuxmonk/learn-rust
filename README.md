## Rust Notes

### Tools and Terminology

- `rustup` is a command line utility to manage installation, upgrades etc of the rust toolchain
- `rustc` is the rust compiler
- `rustfmt` is the source code formatter
- `cargo` is the build tool and package manager
- `crates` is to access functionality outside of the the standard library [https://crates.io](https://crates.io)
- `crate` is a collection of source files
   * `binary crate` is a rust executable
   * `library crate` is a rust library that can be imported by other programs
